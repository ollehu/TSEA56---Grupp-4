== Karta & Avsökning==

map[28][28]
Innehåller information om vilka kartmoduler som roboten besökt, vilka kartmoduler som är klassade som väggar och var start respektive mål är. Räknar upp med utgångspunkt från start.

uint8_t startPositionValue = 0xF2;
uint8_t	targetValue = 0xF3;
uint8_t wallValue = 0xF4;
uint8_t	initialValue = 0xF5;


path[28][28]
Innehåller information om vilka vägar som är ointressanta samt numrerar besökta kartmoduler med utgångspunkt från målet.

uint8_t	targetPathValue = 0;
uint8_t blockedWayValue = 0xF1;
uint8_t	initialValue = 0xF5;


Roboten startar i (14,14) och i nordlig riktning. map och path är utifrån startpositionen och roterar inte med roboten.

Avsökningen sker med hjälp av högerföljning. 

De vägar i path som klassas som ointressanta är följande:
- Återvändsgränd där målet inte hittats
- väg som inte kan ge kortare väg till målet än den pessimistiska skattningen (den första vägen roboten tog till målet)
